# Definir token de autenticacao

Salva o token JWT no `.Renviron` e define na sessao atual.

## Usage

``` r
ficharium_definir_token(token)
```

## Arguments

- token:

  string com o token JWT

## Value

invisivel

## Details

Definir token de autenticacao

## Examples

``` r
if (FALSE) { # \dontrun{
ficharium_definir_token("meu_token_jwt")
} # }
```
