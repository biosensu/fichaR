# Obter um projeto especifico

Retorna os detalhes completos de um projeto pelo seu ID.

## Usage

``` r
projeto(id)
```

## Arguments

- id:

  string com o ID do projeto

## Value

lista R com os campos do projeto

## Details

Obter um projeto especifico

``` r
if (FALSE) { # \dontrun{
projeto("abc123")
} # }
```
